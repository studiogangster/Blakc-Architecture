# Storage Service

## Microservices

This component has a single microservice. **Storage Service**

## Tasks

#### Storage service will be responsible for

* Creating pre-signed URL for S3
* Managing user level authorisation before providing pre-signed URL



## Dependencies

* MongoDB \(Database\)
